🌿 Plant Parent

A custom ServiceNow application built to demonstrate core ServiceNow development concepts through a real-world plant management system.

⸻

📖 Overview

Plant Parent is a custom scoped ServiceNow application that allows users to manage their houseplants by tracking watering schedules, sunlight requirements, plant health, and care instructions.

This project was built as a portfolio application to strengthen hands-on ServiceNow development skills and demonstrate platform knowledge commonly evaluated during junior ServiceNow Developer interviews.

⸻

✨ Features

* 🌱 Manage individual plants
* 🌿 Reusable Plant Care Profiles
* 💧 Water Plant UI Action
* 📅 Automatic Next Watering calculation
* 📝 Health tracking and notes
* ⚡ Instant field population using GlideAjax
* 🔄 Business Rules for server-side automation
* 🛡️ UI Policies for form validation

🎯 Why I Built This Project

I created Plant Parent to gain hands-on experience with ServiceNow development while building a realistic application that demonstrates platform configuration, scripting, database design, automation, and client/server communication.

Throughout the project, I focused on following ServiceNow best practices, minimizing duplicate logic, and creating a maintainable application architecture.
